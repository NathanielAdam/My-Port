Front

Front end technologies are the "View", generally speaking if you look at any website these will be the languages doing the talking. They'll; be colorful and should flow while keeping the User Experience in mind to provide an easy to understand interface.

Angular

Front end framework utilizing Typescript, a superset of Javascript that comes front loaded with tools for routing, smart and lazy renders, and has long term support by google, ensuring that ensures that it will be receiving feature and optimization updates for the forseeable future.

React

React is a front end library, or framework depending on who you ask, that is lightweight and component based, with tools for handling state and routing for a smoothe lightweight applications for both mobile and desktop.
